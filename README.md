# Human Resource Data Analysis

## 📌 Project Overview
This project analyzes Human Resource (HR) employee data to identify patterns related to employee attrition, demographics, and workforce trends. The analysis includes data cleaning, exploration, and visualization to help organizations make data-driven HR decisions.

---

## 🎯 Objectives
- Analyze employee attrition trends.
- Clean and preprocess HR data.
- Explore employee demographics.
- Create visualizations for better insights.
- Build an interactive Power BI dashboard.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/souvik835217-byte/Human_Resource_Data_Analysis.git
```

2. Open the project folder.

3. Open `HR_data.ipynb` in Jupyter Notebook or VS Code.

4. Run all notebook cells.

5. Open `dashboard1.pbix` using Microsoft Power BI Desktop to explore the dashboard.
